Check-in Game API协议

API协议用于支持第三方客户端

首页

签到流程

  • 根据用户名和密码获取签到Token
  • 提交用户名和签到Token

返回数据

  • 数据均以JSON格式返回
返回值例子
{ "body": "example", "code": 200, "msg": "success" }
返回值解释
  • body：返回值主体
  • code：情况编号，200为正常，404为找不到相关信息
  • msg：错误情况的大致描述

获取Token

  • 节点：/getToken.php
参数
  • username：用户名
  • password：密码
案例

如要获取用户abcd123，密码12345678的Token：

  • 提交地址：https://checkin.tcapps.twocola.com/getToken.php?username=abcd123&password=12345678
成功返回值
{body: "4f7dbfa46c0e0ff25db95d0d68f45584@abcd123", code: 200, msg: "success"}
失败返回值
{body: false, code: 404, msg: "Failed to generate token."}

签到

  • 节点：/checkIn.php
参数
  • username：用户名
  • token：签到Token
案例

如要获取用户abcd123，Token为上方获取的4f7dbfa46c0e0ff25db95d0d68f45584@abcd123：

  • 提交地址：https://checkin.tcapps.twocola.com/checkIn.php?username=abcd123&token=4f7dbfa46c0e0ff25db95d0d68f45584
成功返回值
{body: true, code: 200, msg: "success"}
失败返回值
{body: false, code: 404, msg: "Failed to check in."}
